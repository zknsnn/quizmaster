# QuizMaster

Dit project is mijn afstudeerproject voor het Make It Work software engineering traject. 
Het is ontwikkeld als teamproject met Java MVC voor de backend en een frontend met Java FXML.

## Projectbeschrijving
QuizMaster is een educatief platform waarmee docenten klassen kunnen aanmaken, studenten kunnen registreren, examens kunnen toewijzen en studenten hun resultaten kunnen bekijken. 
Het systeem ondersteunt de volgende functionaliteiten:

### Functionaliteiten

**Docent / Beheerder**
- Klassen aanmaken en beheren
- Docenten toewijzen aan klassen
- Studenten registreren en beheren

**Student**
- Registratie en inloggen
- Examens maken
- Resultaten bekijken
